Employment and Wages

Drop down menu for a user to select an industry

Map Visual 1

This section will contain a map of the United States with states you can click on. When a state is selected, it will appear in the center column of this row. The colors of each state will follow a gradient depending on how prevelant the chosen industy is for that state.

Map Visual 2

This section will contain a map of the chosen state with counties colored in the scheme noted in the left hand column. The user can click on each county so that the county name will appear in the visualization in the right column of this row.

County List and Splitter

This section will contain a list of chosen counties and some settings management tools. For example, there will be a multiple selection toggle switch, which will allow the user to select multiple counties to look at. There will also be an option to aggregate selected counties into one group. These toggles will change the displays of the visualizations below.

Wages

Left Column

This visualization will show the average wage for an industry and state over the last twenty years.

Right Column

This visualization will be a line graph displaying the change in average wages over the last ten years.

Employment

Center Column

This visualization will be a line graph for the number of people employed by a selected industry.

Center Column

This visualization will either be a pie chart or a bar graph for the percent of employment in each industry for a selected year.

Center Column

This visualization will be a line graph for the percent change in number of people employed by a selected industry
